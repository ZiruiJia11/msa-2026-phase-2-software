# Agent Instructions

These instructions record the working rules for AI tools used during the FitQuest project.

## Technical Requirements

- Use React with TypeScript for the frontend.
- Use C# with .NET for the backend.
- Use Entity Framework Core for database access.
- Use persistent database storage, not only in-memory storage.
- Use Scalar for API documentation.

## Development Approach

- Keep code simple, readable, and suitable for a student full-stack project.
- Do not claim unfinished features are implemented.
- Explain meaningful code changes when making them.
- Keep the project aligned with the FitQuest concept and gamification theme.
- Prioritise basic requirements before optional features.

## Safety and Scope

- Avoid adding medical advice, personalised diet plans, or professional training recommendations.
- Focus on motivation, habit tracking, XP, streaks, achievements, and progress tracking.
